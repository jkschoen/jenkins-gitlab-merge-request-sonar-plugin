This the template to format the new issues found that will be included in the comment posted. You can use the following variables in the template:

  • $KEY -
  • $COMPONENT - the file that this issue was found in.
  • $LINE - the line in the component where the issue is found
  • $MESSAGE - the issue message
  • $SEVERITY - the severity of the issue
  • $RULE - the rule the issue violates
